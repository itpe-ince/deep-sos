# SOS랩 - 글로컬대학 지역사회문제해결 리빙랩 & 온라인 사회공헌 플랫폼

> 대학 역량 기반 지역혁신을 주도하는 리빙랩 운영 + 온라인 사회공헌 플랫폼(USCP) 구축

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | SOS랩 (지역사회문제해결 리빙랩 + 온라인 사회공헌 플랫폼) |
| **상위 계획** | 2025 글로컬대학 본지정 실행계획서 §4-3 "Local-to-Global 사회 책무성 실현" |

### 1.4 리빙랩 5단계 프로세스 (실행계획서 기반)

| 단계 | 탐색 | 실행 | 개발 | 검증 | 활용 |
|------|------|------|------|------|------|
| **참여주체** | 지역혁신기관, 리빙랩 네트워크 | 지역사회특화센터, 지자체 | 지역사회특화센터, 비즈니스개발본부, 미래창업원, 지역기업 등 | | 지자체, 리빙랩 네트워크 |
| **주요역할** | 지역문제 정의, 수집 및 제공 | 리빙랩 프로젝트 기획 및 수행 | 핵심기술 기반 시(차)제품 등 개발 | 유용성 시험, 피드백 등 평가 | 리빙랩 성과 대내외 확산 |

### 1.5 USCP (Union Social Contribution Platform) 요건

실행계획서에서 명시한 USCP 핵심 요건:
- 대학과 지역사회 간 **협력 강화**와 사회공헌 활동의 **확산**을 위한 온라인 시스템
- **디지털 거버넌스** 기반의 지속 가능한 공공 협력체계 구현
- **사회복지자원봉사인증관리(VMS)** 및 **1365 자원봉사포털** 연계
- 통합대학 간 물리적 제한 극복 및 효과적 사회공헌 활동 운영

## 5. 차별화 포지셔닝

### 5.1 기존 환경 vs SOS랩

```
기존 환경                              SOS랩 해결
━━━━━━━━━━━━━━━━━━━━━━━             ━━━━━━━━━━━━━━━━━━━━━━━
에브리타임 (학생끼리, 비공식)          → 교수-학생-지역 통합 소통
LMS (학사 관리만)                    → 리빙랩 프로젝트 관리 + 소통
봉사활동 (VMS 별도 관리)             → VMS/1365 자동 연계
ESG/SDGs (수기 보고)                → 디지털 기록·추적·대시보드
리빙랩 (오프라인, 파편화)             → 온라인 통합 관리 플랫폼
캠퍼스 간 단절 (4개 캠퍼스)           → 멀티캠퍼스 통합 플랫폼
지역 이슈 (접수 후 방치)             → 투명한 처리 추적 + 결과 환류
```

### 5.2 핵심 가치 제안

> **"지역의 문제를 발굴하고, 대학의 역량으로 해결하며, 성과를 지역에 환원하는 플랫폼"**
> - 리빙랩 5단계 프로세스의 디지털화
> - 대학-지자체-지역기업-시민의 협력 허브
> - 사회공헌 활동의 통합 관리 및 VMS/1365 연계
> - ESG/SDGs 성과의 체계적 기록과 시각화

## 6. 활성화 전략

### 6.1 참여 동기 설계

#### 등급 체계
```
학생: 새내기 → 탐험가 → 해결사 → 메이커 → 리더
교수: 참여교수 → 활동교수 → 멘토교수 → 혁신교수
시민: 관심시민 → 참여시민 → 적극시민 → 지역리더
```

#### 사회공헌 연계 보상
- 리빙랩 참여 → 봉사시간 인정 (VMS 자동 연동)
- 프로젝트 기여 → 사회공헌 포트폴리오 기록
- 우수 활동 → 장학금/인센티브 연계 가능

### 6.2 활성화 전략

| 전략 | 내용 | 기대 효과 |
|------|------|----------|
| **캠퍼스 앰배서더** | 4개 캠퍼스별 SOS랩 홍보 학생 선정 | 초기 사용자 확보 |
| **교수 인센티브** | 사회공헌 활동을 업적평가 반영 | 교수 참여 확보 |
| **지자체 연계** | 대전/공주/예산/세종 지자체 공식 채널 인정 | 이슈 처리 실효성 |
| **봉사시간 인정** | 플랫폼 활동 → VMS 봉사시간 자동 인증 | 학생 참여 동기 |
| **리빙랩 공모전** | 캠퍼스별 리빙랩 프로젝트 경진대회 | 우수 프로젝트 발굴 |
| **SDGs 챌린지** | 월별 SDGs 목표 연계 캠페인 | 지속적 참여 유도 |
| **성과 공유회** | 학기별 리빙랩 성과 발표 + 지역 확산 | 결과 환류, 동기 부여 |

### 6.3 멀티캠퍼스 운영 전략

| 구분 | 통합 운영 | 캠퍼스별 분리 |
|------|---------|------------|
| **통합** | 인증(SSO), 사회공헌 포트폴리오, ESG 대시보드, 성공사례 | |
| **분리** | | 리빙랩 프로젝트 (캠퍼스 특화), 지역 이슈 (지역별), 커뮤니티 |
| **연계** | 캠퍼스 간 협업 프로젝트, 성과 공유, 노하우 이전 | |
