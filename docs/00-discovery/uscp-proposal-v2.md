# USCP 플랫폼 구축 사업계획서 V2.2

> **수신**: 국립공주대학교 글로컬사업단 지역사회특화센터<br/>
> **사업명**: USCP (Union Social Contribution Platform / 온라인 사회공헌 플랫폼) 구축<br/>
> **근거 문서**: 한혜진 연구원 작성 USCP 플랫폼 구축 계획(안) (2026-04-29) + 1차 실무 미팅 결정 사항 (2026-05-11) + 2026-05-14·15 추가 결정<br/>
> **작성일**: 2026-05-15<br/>
> **버전**: V2.2 (공급가 5,000만 원 한도 + VAT 별도 + Function Point 기반 실 산정, 봉사활동 기능 전면 제외)

---

## 1. 사업 개요

### 1.1 사업 목적

국립공주대학교 글로컬사업단 지역사회특화센터는 2025 글로컬대학 본지정 실행계획서 추진과제 4-3-1에 따라 **온라인 사회공헌 플랫폼(USCP)** 을 구축합니다. 본 플랫폼은 5개 지역(대전·공주·예산·천안·세종)의 지역사회 문제를 시민이 직접 제보하고, 지역사회특화센터의 게이트키핑을 거쳐 리빙랩 의제로 발전시키며, 멘토단(지자체·출연연·기업)과 사회혁신가 양성과정 학생팀이 협력하여 해결하는 6단계 의제 라이프사이클을 디지털화합니다.

### 1.2 사업 배경

본 사업은 다음 두 단계의 의사결정을 종합 반영합니다.

| 시점 | 내용 |
|------|------|
| 2026-04-29 | 한혜진 연구원 USCP 플랫폼 구축 계획안 제출 — 시민 참여형 6단계 운영 구조, 5개 지역 양교 협력 체계 정립 |
| 2026-05-11 | 1차 실무 미팅 — 회원 제보 모델 전환, CMS 부분 도입, 멘토단 관리자 주도 매칭, 충남대 단순 협력 기관 정책 등 28건의 핵심 사항 확정 |
| 2026-05-14 | 추가 결정 — (1) 구축비 + 1~3년차 유지보수비 통합 산정 (2) 봉사활동 관련 기능(VMS·1365 자원봉사포털 연계 포함) 본 사업 범위 전면 제외 |
| 2026-05-15 | 추가 결정 — (1) 총 예산 한도 5,000만 원은 **공급가 기준** (VAT 별도) (2) 5,000만 원 이하 산정 허용 (3) **Function Point 기반 실 산정** 적용 (NIPA SW사업 대가산정 가이드 2024 + IFPUG 4.3 표준) (4) **대학 전산실 인프라 사용 여부 미정** (RFP 단계 결정 — 미사용 시 클라우드·상용 호스팅·IDC 임대 등 옵션별 가산 별도 산정) |

본 사업계획서는 위 결정 사항을 종합 반영한 V2.2 산출물이며, 향후 RFP 작성 시 본 문서가 기준 자료가 됩니다.

### 1.3 운영 원칙

본 사업은 **실무자 시점에서 관리 기능의 충실성**을 우선 가치로 삼습니다.

1. **충실한 관리 도구 제공**: 지역사회특화센터 운영자가 일을 효율적으로 수행할 수 있는 명확한 워크플로우와 도구를 우선
2. **법적 의무 충족**: 개인정보보호법·정보통신망법·접근성 등 법령상 의무를 누락 없이 반영
3. **데이터 자산화**: 1차년도 운영 데이터가 2027년 이후 고도화의 기반이 되도록 구조 설계
4. **단순 명확한 권한 모델**: 지역사회특화센터 운영자 단일 역할로 운영 (복잡한 다단계 권한 매트릭스 미적용)

> 본 사업의 설계 원칙에서 **운영 인력 규모(1인/다수)는 가정하지 않습니다.** 시스템은 사용자(시민 회원·운영자·멘토·학생팀) 관점에서 필요한 기능을 제공하는 데 집중하며, 운영 인력 규모는 발주처의 결정 영역입니다.

### 1.4 단계별 추진 전략

| 단계 | 기간 | 포커스 | 핵심 내용 |
|------|:----:|------|----------|
| **구축 (1차년도)** | 2026년 (분석설계 2~3개월 + 개발 5~6개월 + 시범운영 2개월) | 핵심 플랫폼 구축 + 시범운영 | 6단계 의제 라이프사이클·로그인 회원 제보·게이트키핑·5개 지역 운영·멘토단 매칭·이메일 알림 |
| **유지보수 1년차** | 2026.12~2027.12 (무상 포함) | 안정화 | 소프트웨어 버그 수정·서버 장애 대응·경미한 UI 수정 |
| **유지보수 2~3년차** | 2027.12~2029.12 (유상 Basic) | 운영 지원 | 장애 대응·정기 점검·소규모 요청 처리 |

> 본 사업은 **봉사활동 관련 기능(VMS·1365 자원봉사포털 연계, 봉사실적·시간 관리 등)을 전면 제외**하며, 5,000만 원 한도 내 구축비 + 1~3년차 유지보수비 통합 산정 구조를 채택합니다 (2026-05-14 발주처 결정).

### 2.2 사용자 구조

| 역할 | 대상 | 인증 | 주요 활동 |
|------|------|------|----------|
| **시민 회원** | 5개 지역 일반 시민·지역주민 | 이메일 회원가입 (만 14세 이상, 개인정보·약관 동의 필수) | 지역문제 제보·공감투표·댓글·진행 상황 조회 |
| **지역사회특화센터 운영자** | 발주처 실무 담당자 (단일 역할) | 이메일 로그인 | 게이트키핑(제보 승인/반려)·리빙랩 등록·멘토 선정·콘텐츠 관리·성과지표 입력 |
| **멘토** | 지자체·출연연·기업 협의된 전문가 (가입된 사용자 중 운영자가 멘토 자격 부여) | 이메일 회원가입 후 운영자가 멘토 자격 부여 | 의제별 멘토단 활동·산출물 검토 |
| **학생팀** | 공주대·충남대 사회혁신가 양성과정 학생 (가입된 사용자 중 운영자가 직접 선정·구성) | 이메일 회원가입 후 운영자가 팀 배정 | 의제별 리빙랩 실행·산출물 업로드 |

> **양교 협력 (1차 미팅 신규)**: 충남대학교는 MOU 체결된 협력 기관이며, **특별한 시스템 권한·연동 기능 없이 단순 협력 기관**으로 운영됩니다. 충남대 학생·교수도 일반 시민 회원과 동일한 절차로 회원가입합니다. 데이터 소유권은 공주대 단독입니다.

### 2.4 사용자 영역 (시민 회원 로그인 후)

> **[1차 미팅 변경]**: 한혜진 계획안의 비로그인 시민 자유 제보 모델이 **로그인 회원 제보 모델**로 변경되었습니다. 회원가입 시 개인정보 수집·이용 동의와 이용약관 동의가 통합 처리되며, 만 14세 이상만 이용 가능합니다.

```
시민 회원 영역
├── [1] 회원가입 / 로그인
│   ├── 이메일·비밀번호 가입
│   ├── 개인정보 수집·이용 동의 (필수)
│   ├── 이용약관 동의 (필수)
│   ├── 만 14세 이상 확인 (약관 명시 우회)
│   └── 이메일 검증
├── [1] 지역문제 제보
│   ├── 5개 지역 선택
│   ├── 제목·내용 입력
│   ├── 사진 첨부 (개수·용량 제한 RFP 단계 확정)
│   └── 제보 등록 완료 → 검토 단계 진입
├── [1] 공감투표 (로그인 사용자 1인 1회)
├── [1] 댓글 작성 (로그인 필수)
├── [1] 내 제보 진행 상황 조회
├── [1] 이메일 알림 수신 (단계 변경 시마다 — 1차 미팅 결정)
└── [1] 프로필 설정 (이름·소속·연락처)
```

### 2.5 관리자 영역 (지역사회특화센터 운영자 단일 역할)

> **[1차 미팅 변경]**: 슈퍼/운영/일반관리자 3단계 역할이 **지역사회특화센터 운영자 단일 역할**로 단순화되었습니다. 메뉴×역할 권한 매트릭스 미적용, 게이트키핑 권한은 모든 운영자에게 부여됩니다.
