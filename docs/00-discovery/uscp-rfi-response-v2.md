## 2. 발주처 RFI 이해도

### 2.1 사업 배경 및 의사결정 과정

발주처는 다음 두 단계의 의사결정을 거쳐 본 사업의 범위·운영 모델을 확정하였습니다.

#### 2.1.1 한혜진 연구원 USCP 플랫폼 구축 계획안 (2026-04-29)

- 5개 지역 (대전·공주·예산·천안·세종, 양교 캠퍼스 기준)
- 4대 구성원 (지·산·학·관)
- 시민 참여형 6단계 의제 라이프사이클 (제보 → 검토중 → 공개등록 → 멘토배정 → 처리중 → 해결완료)
- 멘토단(지자체·출연연·기업) + 사회혁신가 양성과정 학생팀 3자 매칭
- 카카오맵 API 기반 지역별 현황 지도
- 글로컬대학 성과지표 연동
- ~~VMS·1365 자원봉사포털 연계 (중장기)~~ → **2026-05-14 발주처 결정으로 봉사활동 기능 전면 제외**

#### 2.1.2 1차 실무 미팅 결정 사항 (2026-05-11)

발주처와 당사는 1차 실무 미팅을 통해 한혜진 계획안의 운영 정책·기능 범위·비용을 구체화하는 28건의 핵심 결정을 확정하였습니다.

| 영역 | 핵심 결정 |
|------|---------|
| 운영 모델 | 비로그인 시민 자유 제보 → **로그인 회원 제보**로 전환 |
| 인증 | 회원가입 필수, 만 14세 이상, 광고성 정보 발송 없음, 양교 SSO 미적용 |
| 알림 | 이메일 단일 채널, 모든 단계 변경 시 발송, 운영자 일괄 알림 |

| 멘토단 | 가입자 중 운영자 선정, 수동 매칭, 통보형 (수락·거절 절차 없음) |
| 권한 | 지역사회특화센터 운영자 단일 역할, 매트릭스 미적용 |

| 트래픽 | 최대 100명 미미한 수준 |
| AI 자동화 | 본 사업 범위 외 |

## 3. 당사 제안 방향

### 3.1 구축 철학

> **"실무자가 일을 효율적으로 수행할 수 있는 충실한 관리 도구 + 시민이 쉽게 참여할 수 있는 회원 제보 모델"**

본 사업의 설계 원칙에서 **운영 인력 규모는 가정하지 않습니다.** 시스템은 사용자(시민 회원·운영자·멘토·학생팀) 관점에서 필요한 기능을 제공하는 데 집중하며, 인력 규모는 발주처의 결정 영역입니다.

- **충실한 관리 도구**: 게이트키핑·리빙랩 운영·멘토단 매칭·콘텐츠 관리에 필요한 기능을 풍부하게 제공
- **법적 의무 준수**: 개인정보·약관·감사 로그·정보통신망법 요건을 누락 없이 반영
- **단순 명확한 권한**: 지역사회특화센터 운영자 단일 역할 (복잡한 권한 매트릭스 미적용)
- **데이터 자산화**: 구축 단계 운영 데이터가 1~3년차 유지보수 기간 동안 안정적으로 누적되도록 구조 설계

### 3.2 단계별 추진 전략

| 단계 | 기간 | 포커스 |
|------|:----:|------|
| **구축 (1차년도)** | 2026년 (분석설계 2~3개월 + 개발 5~6개월 + 시범운영 2개월) | 핵심 플랫폼 + 시범운영 |
| **유지보수 1년차 (무상)** | 2026.12~2027.12 | 안정화·버그 수정·서버 장애 대응 |
| **유지보수 2~3년차 (유상 Basic)** | 2027.12~2029.12 | 장애 대응·정기 점검·소규모 요청 처리 |

> **봉사활동 관련 기능(VMS·1365 자원봉사포털 연계 포함)은 2026-05-14 발주처 결정으로 본 사업 범위에서 전면 제외**되었으며, 본 사업은 **5,000만 원 한도 내 구축비 + 1~3년차 유지보수비 통합 산정** 구조를 채택합니다.

### 3.3 데이터 흐름 (논리)

```
시민 회원 가입 + 동의
        ↓
지역문제 제보 (1단계)
        ↓
운영자 게이트키핑 (이메일 일괄 알림)
        ↓
2단계 검토중 / 3단계 공개등록 / 4단계 멘토배정 / 5단계 처리중
        ↓
6단계 해결완료 → 성과지표 자동 카운트 + 성공사례 4단계 스토리 작성
        ↓
글로컬대학 성과지표 연동 + 활동 타임라인 아카이빙
```

## 4. 플랫폼 전체 구조 (Information Architecture)

### 4.1 사용자 구조

| 역할 | 인증 | 주요 권한 |
|------|------|---------|
| 시민 회원 | 이메일 회원가입 (만 14세 이상 + 개인정보·약관 동의) | 제보 등록·공감투표·댓글·내 활동 조회 |
| 지역사회특화센터 운영자 | 이메일 로그인 (단일 역할) | 게이트키핑·리빙랩 등록·멘토 선정·콘텐츠 관리·성과지표 입력 |
| 멘토 | 회원가입 후 운영자가 멘토 자격 부여 | 의제별 멘토단 활동·산출물 검토 |
| 학생팀 | 회원가입 후 운영자가 팀 배정 | 의제별 리빙랩 실행·산출물 업로드 |

### 4.2 공개 영역 (6개 메뉴)

```
1. 홈
2. USCP 소개 (정적 HTML)
   - 플랫폼 소개 / 5개 지역 안내 / 참여주체 안내 / 양교 MOU 안내 (신규)
3. 지역문제 광장
4. 리빙랩 (목록·상세·활동 타임라인·산출물 DB·성공사례)
5. 협력 네트워크 (참여기관·MOU·프로그램·커뮤니티)
6. 성과자료 (성과지표·공지사항·자료실)
```

### 4.3 시민 회원 영역

```
- 회원가입·로그인
- 지역문제 제보 (5개 지역·사진 첨부)
- 공감투표 (1인 1회 — 로그인 사용자만)
- 댓글 (로그인 필수)
- 내 제보 진행 상황 조회
- 이메일 알림 수신 (모든 단계 변경 시)
- 프로필 설정
```

### 4.4 관리자 영역

```
- 대시보드 (현황 수치·처리 대기·차트)
- 게이트키핑 (6단계 전환·승인·반려·검토 의견)
- 리빙랩 관리 (등록·활동 타임라인·산출물·성공사례 4단계)
- 멘토단 운영 (멘토 자격 부여·수동 매칭·학생팀 구성)
- 협력 네트워크 (협력기관·MOU·프로그램·커뮤니티)
- 성과자료 (성과지표·실적 입력·자동 집계)
- 콘텐츠 관리 (공지·이벤트 WYSIWYG / 자료실 / 배너 / 약관 WYSIWYG)
- 권한·감사 (운영자 관리·로그·약관 버전)
```
