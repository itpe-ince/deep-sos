# USCP 2차년도 기능 목록

> **프로젝트**: 온라인 사회공헌 플랫폼 (USCP)<br/>

## 1. Information Architecture (2차년도 추가분)

> 1차년도 IA에 아래 메뉴가 추가됩니다. **[신규]** 표시는 새로 추가되는 메뉴, **[확장]** 표시는 기존 메뉴에 기능이 추가되는 항목입니다.

### 1.1 공개 영역 추가

```
USCP
├── ...기존 메뉴 유지...
├── 리빙랩 프로젝트
│   └── 프로젝트 지원 도구 강화
│       ├── ...기존 탭 유지...
│       └── [확장] 마일스톤
├── 성공사례
│   └── [확장] 성공사례 상세
├── [신규] AI 챗봇 (전체 화면)
└── 로그인 / 회원가입
    └── [확장] 소셜 로그인 (카카오/네이버/구글)
```

### 1.2 사용자 영역 추가

```
내 정보
└── [신규] 내 대시보드
    ├── 내 참여 프로젝트
    ├── 내 산출물
    └── 내 알림
```

### 1.3 관리자 영역 추가

```
관리자
├── 대시보드
│   ├── [확장] 시각화 차트 (진행률/캠퍼스별/월별)
│   └── [확장] 미처리 항목 뱃지
├── 프로젝트 관리
│   ├── [신규] 프로젝트 칸반 보드 (3단계/5단계 통합)
│   ├── [확장] 프로세스 모드 선택 (3단계/5단계)
│   ├── [확장] 마일스톤 관리
│   ├── [확장] 단계별 산출물 분류
│   ├── [확장] 팀별 산출물 현황
│   └── [확장] 사업화 연계 이력 관리
├── 이슈 관리
│   ├── [확장] 이슈→프로젝트 원클릭 전환
│   ├── [확장] 상태 변경 타임라인
│   └── [확장] 이슈 통계 차트
├── 기관·MOU 관리
│   ├── [확장] MOU 만료 자동 알림
│   └── [확장] 기관별 프로젝트 현황
├── 콘텐츠 관리
│   ├── [확장] 스마트 에디터
│   ├── [확장] 배너 게시 기간 자동 관리
│   └── [확장] 페이지 발행 예약 / 수정 이력
├── [신규] 성공사례 관리 (전용)
│   └── 4단계 스토리 편집
├── 사용자 관리
│   └── [확장] 사용자 상세 (참여 프로젝트/활동 이력)
├── [신규] AI 챗봇 관리
│   ├── 대화 로그 조회
│   ├── 미응답 질문 관리
│   ├── 응답 피드백 현황
│   ├── 데이터 소스 관리
│   ├── 인덱싱 상태 모니터링 (마지막 동기화, 문서/청크 수, 실패 항목)
│   ├── 수동 재인덱싱 실행
│   ├── 외부 문서 업로드 (PDF, HWP 등)
│   ├── 시스템 프롬프트 편집
│   └── API 사용량 모니터링
└── 시스템 설정 (시스템관리자 전용)
    ├── [확장] 프로젝트별 역할 관리 (프로젝트 관리자/팀장/팀원)
    ├── [확장] 약관 변경 시 재동의 관리
    ├── [신규] 배치/스케줄러 관리
    │   ├── 배치 작업 목록 (RAG 인덱싱, MOU 만료 체크, 배너 자동 관리)
    │   ├── 스케줄 설정 (실행 주기, 활성화/비활성화)
    │   └── 실행 결과 조회 (마지막 실행, 성공/실패, 처리 건수)
    └── [신규] 시스템 설정 관리
        ├── 사이트 기본 정보 (사이트명, 로고, 대표 이메일, 기관 정보)
        ├── 이메일 발송 설정 (SMTP 서버, 발신자, 발송 활성화)
        ├── 파일 업로드 설정 (허용 확장자, 최대 용량, 저장 경로)
        ├── 보안 설정 (세션 타임아웃, 로그인 실패 제한, 비밀번호 정책)
        ├── 알림 설정 (알림 유형별 활성화/비활성화, 발송 채널)
        ├── 캠퍼스 관리 (캠퍼스 목록, 순서, 활성화)
        └── 환경 변수 관리 (외부 API 키, 시스템 파라미터 — 재시작 없이 적용)
```

### 1.4 공통 추가

```
[신규] 실시간 알림 (헤더 벨 아이콘)
├── 알림 드롭다운 목록
└── 읽음/안읽음 관리

[신규] 통합 검색 (헤더 검색창)

[신규] AI 챗봇 위젯 (전 페이지 하단 우측)
```

| SYS-01 | 사이트 기본 정보 | 사이트명, 로고 이미지, 대표 이메일, 기관 정보, 저작권 문구 등. 1차에서는 코드에 고정, 2차에서 UI 관리 |
| SYS-02 | 이메일 발송 설정 | SMTP 서버 주소, 포트, 발신자 이메일, 발송 활성화/비활성화. 비밀번호 재설정, 알림 발송에 사용 |

### 2.11 AI 챗봇 — RAG 파이프라인 / 데이터 운영

|---|------|------|
| A-07 | 문서 수집기 (Crawler) | 플랫폼 DB에서 인덱싱 대상 데이터를 추출. CMS 페이지, 프로젝트 정보, 성공사례, FAQ, 공지사항 등 |
| A-08 | 문서 전처리 | 수집된 데이터를 청크(chunk) 단위로 분할. 메타데이터(출처 URL, 카테고리, 수정일) 부착 |

| A-13 | 임베딩 버전 관리 | 재인덱싱 시 이전 버전 보관. 문제 발생 시 이전 버전으로 롤백 가능 |
