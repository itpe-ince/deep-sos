# USCP 2차년도 기능 목록

> **프로젝트**: 온라인 사회공헌 플랫폼 (USCP)<br/>
> **사업 연도**: 2차년도 (3,000만원 VAT 포함)<br/>
> **전제조건**: 1차년도 플랫폼 구축 완료<br/>
> **작성일**: 2026-04-21

---

## 1. Information Architecture (2차년도 추가분)

> 1차년도 IA에 아래 메뉴가 추가됩니다. **[신규]** 표시는 새로 추가되는 메뉴, **[확장]** 표시는 기존 메뉴에 기능이 추가되는 항목입니다.

### 1.1 공개 영역 추가

```
USCP
├── ...기존 메뉴 유지...
├── 성공사례
│   └── [신규] 성공사례 상세 (4단계 스토리)
├── [신규] JA 교원 소개 (공개 프로필 목록)
└── 로그인 / 회원가입
    └── [확장] 소셜 로그인 (카카오/네이버/구글)
```

### 1.2 사용자 영역 추가

```
내 정보
└── [신규] 내 대시보드
    ├── 내 참여 프로젝트
    ├── 내 산출물
    └── 내 알림
```

### 1.3 관리자 영역 추가

```
관리자
├── 대시보드
│   ├── [확장] 시각화 차트 (현황/캠퍼스별/월별)
│   └── [확장] 미처리 항목 뱃지
├── 프로젝트 관리
│   ├── [확장] 프로젝트 권한 관리 (프로젝트 관리자/팀장/팀원/열람자)
│   └── [확장] 사업화 연계 이력 관리
├── 이슈 관리
│   ├── [확장] 이슈→프로젝트 원클릭 전환
│   ├── [확장] 상태 변경 타임라인
│   └── [확장] 이슈 통계 차트
├── 기관·MOU 관리
│   ├── [확장] MOU 만료 자동 알림
│   └── [확장] 기관별 프로젝트 현황
├── [신규] CMS (콘텐츠 관리)
│   ├── 소개 페이지 관리 (스마트 에디터)
│   ├── 배너 관리 (게시 기간 자동)
│   ├── 성공사례 관리 (4단계 스토리)
│   └── 약관 관리 (개인정보처리방침/이용약관)
├── [신규] JA 교원 관리
│   ├── JA 교원 등록 (내·외부 구분)
│   ├── 위촉 계약 관리
│   └── 활동 실적 조회
├── [신규] 리빙랩 교과목 관리
│   ├── 교과목 등록
│   ├── 팀별 문제분석·중간보고·최종발표
│   ├── 교수자·멘토 평가
│   ├── 학생 상호평가
│   ├── 성찰일지
│   ├── 우수성과 공유회
│   └── 우수방안 채택·활용 기록
├── [신규] KPI 관리
│   ├── 지표 등록 (연도별 목표치)
│   ├── 실적 입력 (월별/분기별)
│   └── 목표 대비 달성률 대시보드
├── [신규] 거버넌스
│   ├── 협의체 회의록
│   ├── 의사결정 이력
│   └── 정책 피드백 추적
├── 사용자 관리
│   └── [확장] 사용자 상세 (참여 프로젝트/활동 이력)
└── 시스템 설정 (시스템관리자 전용)
    ├── [확장] 약관 변경 시 재동의 관리
    └── [신규] 시스템 설정 관리
        ├── 사이트 기본 정보 (사이트명, 로고, 대표 이메일)
        ├── 이메일 발송 설정 (SMTP)
        ├── 파일 업로드 설정 (확장자, 용량)
        ├── 보안 설정 (세션, 비밀번호 정책)
        ├── 알림 설정 (유형별 활성화, 채널)
        ├── 캠퍼스 관리 (목록, 순서, 활성화)
        └── 환경 변수 관리 (API 키, 재시작 없이 적용)
```

### 1.4 공통 추가

```
[신규] 실시간 알림 (헤더 벨 아이콘)
├── 알림 드롭다운 목록
└── 읽음/안읽음 관리

[신규] 통합 검색 (헤더 검색창)
```

> **참고**: 프로젝트 5단계 프로세스 관리(칸반/마일스톤), AI 챗봇, AI 기반 지역문제 자동 수집, AI 산출물 분석은 별도 제안서에 포함됩니다.

## 3. 별도 제안 사항

아래 기능들은 본 문서 범위에 포함되지 않으며, **별도 제안서**에 기재됩니다.

| 카테고리 | 주요 기능 |
|---------|----------|
| 프로젝트 관리 고도화 | 5단계 프로세스(탐색→실행→개발→검증→활용), 칸반 보드, 마일스톤 관리, 기한 알림 |
| AI 챗봇 | RAG 파이프라인, LLM 연동, 챗봇 UI, 대화 로그 관리 |
| AI 지역문제 자동 수집 | 외부 뉴스/민원 크롤러, LLM 자동 분류, 이슈 후보 관리 |
| AI 산출물 분석 자동화 | 산출물 자동 분석, 성과 평가 리포트, 승인 워크플로우 |
