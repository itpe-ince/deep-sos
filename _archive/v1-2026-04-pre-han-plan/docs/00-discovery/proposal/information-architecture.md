# USCP 통합 Information Architecture

> **프로젝트**: 온라인 사회공헌 플랫폼 (USCP)<br/>
> **범위**: 1차년도 + 2차년도 + 별도 제안 통합 IA<br/>
> **작성일**: 2026-04-21

---

## 범례

| 표기 | 의미 |
|------|:---:|
| **[1]** | 1차년도 구축 (5,000만원) |
| **[2]** | 2차년도 구축 (3,000만원) |
| **[별]** | 별도 제안 (3,000만원) |
| **[2→]** | 2차년도 확장 (기존 1차 메뉴 내 기능 추가) |

## 5. 연차별 요약

### 5.1 연차별 범위 요약

| 구분 | 화면 수 | 기능 수 | 총액 (VAT 포함) |
|------|:-------:|:------:|:--------------:|
| **1차년도** | 19개 | 79개 | 5,000만원 |
| **2차년도** | +14개 | +59개 | 3,000만원 |
| **별도 제안** | +9개 | +55개 | 3,000만원 |
| **합계** | **42개** | **193개** | **11,000만원** |

### 5.2 연차별 핵심 포커스

| 구분 | 포커스 | 핵심 키워드 |
|------|------|------------|
| **1차년도** | 비용 최적화 + 핵심 기능 | 프로젝트 관리, 이슈, 기관/MOU, 커뮤니티, 감사 로그, 메뉴/권한 관리 |
| **2차년도** | 운영 편의성 + 센터 체계 | CMS 신규, JA 교원 관리, 리빙랩 교과목, KPI, 거버넌스, 시스템 설정 UI |
| **별도 제안** | AI 자동화 + 지원 | AI 프로젝트 지원, AI 챗봇, 지역문제 자동 수집, 산출물 AI 분석 |

### 5.3 역할별 접근 범위

| 역할 | 1차년도 | 2차년도 추가 | 별도 제안 추가 |
|------|---------|-------------|--------------|
| **시스템관리자** | 메뉴/권한/감사 로그 | + 약관 재동의 + 시스템 설정 UI | + 배치/스케줄러 |
| **관리자** | 프로젝트/이슈/기관/CMS/커뮤니티/사용자 | + CMS + JA 교원 + 리빙랩 교과목 + KPI + 거버넌스 | + AI 챗봇 관리 + 지역문제 자동 수집 + 산출물 분석 |
| **교수/학생** | 산출물 업로드, 게시판 | + 프로젝트별 역할 (관리자/팀장/팀원) + 내 대시보드 | — |
| **시민** | 게시판, 공개 조회 | + 성공사례 상세 + 소셜 로그인 | + AI 챗봇 이용 |
