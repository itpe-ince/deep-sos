## 1. 사업 개요

### 1.1 사업 목적

1차년도에 구축된 USCP 플랫폼을 고도화합니다. 1차년도에 정적으로 제공되던 **소개 페이지를 관리자가 직접 편집할 수 있는 콘텐츠 관리(CMS) 체계**를 신규 구축하고, 1차년도에 누락된 **리빙랩 교과목 운영, JA 교원 관리, 성과 지표 관리, 거버넌스 체계**를 추가하여 지역사회특화센터의 실제 운영 업무를 체계적으로 지원합니다.

### 1.2 사업 범위

2차년도는 크게 세 파트로 구성됩니다.

- **파트 A. 운영 편의성 강화** — 자동화·시각화·알림·소셜 로그인·CMS 신규 구축
- **파트 B. 지역사회특화센터 운영 체계** — 리빙랩 교과목 운영, JA 교원 관리, KPI, 거버넌스 (실행계획서 기반 신규 추가)
- **파트 C. 시스템 설정 관리** — 1차년도에 하드코딩된 설정을 UI에서 관리

> **참고**: AI 챗봇 및 프로젝트 5단계 프로세스 관리는 별도 제안서에 포함됩니다.

## 2. 파트 A. 운영 편의성 강화

### 2.1 소셜 로그인

1차년도의 이메일 자체 로그인에 소셜 로그인을 추가하여 사용자의 가입/로그인 허들을 낮춥니다.

| 기능 | 설명 |
|------|------|
| 카카오 로그인 | 카카오 계정으로 간편 로그인. 별도 회원가입 없이 카카오 인증으로 자동 회원 생성 |
| 네이버 로그인 | 네이버 계정으로 간편 로그인 |
| 구글 로그인 | 구글 계정으로 간편 로그인 |
| 계정 연동 | 기존 이메일 계정에 소셜 계정을 연결하여 복수 로그인 수단 제공 |

### 2.2 프로젝트 권한 관리

1차년도에는 시스템 관리자만 프로젝트를 관리했지만, 2차년도에는 프로젝트 참여자에게 역할별 권한을 부여하여 **프로젝트 참여자가 직접 해당 프로젝트를 관리**할 수 있도록 합니다.

| 프로젝트 역할 | 권한 | 설명 |
|-------------|------|------|
| **프로젝트 관리자** | 프로젝트 설정/수정, 팀 구성, 상태 전환 | 해당 프로젝트의 운영 전반을 담당하는 참여자 (교수 등) |
| **팀장** | 팀 멤버 관리, 산출물 등록/삭제, 게시판 글 작성 | 소속 팀의 결과물 제출과 멤버 관리 |
| **팀원** | 산출물 등록, 게시판 글 작성, 조회 | 참여 활동과 결과물 업로드 |
| **열람자** | 공개 정보 조회만 가능 | 프로젝트에 배정되지 않은 일반 사용자 |

- 시스템 관리자(센터 실무자)는 모든 프로젝트에 대해 전체 권한을 가짐
- 프로젝트 관리자는 시스템 관리자가 지정

### 2.3 이슈 관리 고도화

| 기능 | 설명 |
|------|------|
| 이슈→프로젝트 원클릭 전환 | 이슈 상세 페이지에서 원클릭으로 해당 이슈를 기반으로 프로젝트를 자동 생성 |
| 상태 변경 타임라인 | 이슈의 상태가 변경될 때마다 자동으로 이력이 기록되어 타임라인으로 표시 |
| 카테고리별/상태별 통계 | 관리자 화면에서 이슈 현황을 차트로 시각화 |

### 2.4 기관/MOU 고도화

| 기능 | 설명 |
|------|------|
| MOU 만료 자동 알림 | MOU 종료일 30일 전에 관리자에게 자동 알림 발송 |
| 기관별 프로젝트 현황 | 기관 상세 페이지에서 해당 기관이 참여 중인 프로젝트 목록을 자동 조회 |
| 사업화 연계 이력 | 텍스트 메모 → 연계 사업명, 기관, 연계 시기, 상태를 구조화된 데이터로 관리 |

### 2.5 콘텐츠 관리 (CMS) 신규 구축

1차년도에 정적 페이지로 제공되던 소개 페이지, 배너, 성공사례, 약관을 **관리자가 직접 편집할 수 있는 CMS 체계**로 신규 구축합니다. 발주처가 개발사에 의존하지 않고 콘텐츠를 자체 운영할 수 있게 됩니다.

#### 소개 페이지 CMS

| 기능 | 설명 |
|------|------|
| 소개 페이지 목록 관리 | 관리 대상 페이지(about/guide/campus/FAQ 등) 목록 조회. 신규 페이지 추가/삭제 |
| 스마트 에디터 | 비개발자인 관리자가 직접 페이지 내용을 편집. 기본 서식, 파일 첨부, 이미지 삽입, YouTube 임베드, 이모지 + **동영상 직접 업로드, SNS 임베드, 지도 삽입, 표 편집, 드래그 앤 드롭 단락 순서 조절** 지원 |
| 페이지 발행/비발행 | 공개 상태 관리 (발행/임시저장) |
| 페이지 발행 예약 | 수정 후 예약 시간에 자동 발행 |
| 페이지 수정 이력 | 편집 이력 자동 기록 (누가, 언제, 어떤 내용을 수정) |
| 정적 페이지 마이그레이션 | 1차년도 정적 페이지 콘텐츠를 CMS로 이관 |

## 3. 파트 B. 지역사회특화센터 운영 체계

2025 글로컬대학 본지정 실행계획서에 명시된 지역사회특화센터의 핵심 운영 요소를 플랫폼에 반영합니다.

### 3.1 JA(Joint Appointment) 교원 관리

실행계획서상 **2029년까지 JA 교원 1,000명 확보** 목표. 내·외부 전문가를 체계적으로 관리합니다.

| 기능 | 설명 |
|------|------|
| JA 교원 등록 | 내·외부 구분(교원/지자체 실무자/기업 실무자/지역 전문가), 소속 기관, 전문 분야 |
| 위촉 계약 관리 | 위촉 기간, 역할, 활동 분야, 관련 문서 첨부 |
| JA 활동 실적 | 참여 프로젝트, 멘토링 이력 자동 집계 |
| JA 공개 프로필 | 전문 분야 홍보, 프로젝트 참여 이력 공개 페이지 |

### 3.2 리빙랩 교과목 운영

실행계획서 p.49 "지역문제해결을 위한 리빙랩 프로젝트 교과목 운영"에 대응하는 기능입니다. 리빙랩 프로젝트가 단순 팀 활동이 아니라 **정식 교과목(학점 연계)** 으로 운영되는 경우를 지원합니다.

| 기능 | 설명 |
|------|------|
| 리빙랩 교과목 등록 | 교과목명, 담당 교수(JA 교원), 학기, 수강 인원, 연계 프로젝트 |
| 팀별 문제분석 보고서 | 분석 보고서 업로드 + 피드백 기능 |
| 중간보고 / 최종 발표 | 보고서 업로드 + 발표 자료 + 일정 관리 |
| 교수자·멘토 평가 | 교수자/지역전문가 평가 입력 (점수 + 코멘트) |
| 학생 상호평가 | 팀원 상호평가 폼 + 집계 |
| 성찰일지 | 학생 개인 성찰일지 작성·제출 |
| 우수성과 공유회 | 우수 성과 발표회 일정·참여자·자료 관리 |
| 우수방안 채택·활용 기록 | 채택된 해결안 + 실제 지역/정책 적용 추적 |

### 3.3 성과 지표(KPI) 관리

| 기능 | 설명 |
|------|------|
| KPI 지표 등록 | 지표명, 단위, 연도별 목표치 (예: 융합과정 이수율, JA 교원 수, 마이크로디그리 과정 수) |
| 실적 입력 | 월별/분기별 실적 수동 입력 |
| 자동 집계 지표 | 리빙랩 프로젝트 수, 참여 기관 수, JA 교원 수 등 플랫폼 내 데이터 자동 집계 |
| 목표 대비 달성률 | 연차별 목표값 vs 실적 비교 대시보드 |

> **참고**: 개인별 KPI는 개인정보 이슈로 제외하고, **기관/프로그램 단위** 집계만 수행합니다.

### 3.4 거버넌스 회의록·의사결정 관리

| 기능 | 설명 |
|------|------|
| 협의체 회의록 관리 | 회의 일시, 참석자, 안건, 결정사항 기록 |
| 의사결정 이력 | 프로젝트·정책별 의사결정 타임라인 |
| 정책 피드백 구조 | 프로젝트 성과 → 정책 제안 → 반영 여부 추적 |
